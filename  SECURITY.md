# Security Policy

If you discover a security vulnerability in Sourceplane MCP, please report it privately and responsibly.

Please do not open public GitHub issues for vulnerabilities involving:

- path traversal bypasses
- filesystem isolation bypasses
- provider credential exposure
- blocklist bypasses
- binary detection bypasses
- unauthorized file access
- provider authentication weaknesses

This helps prevent accidental exposure for users before a fix is available.

## Reporting

Please contact:

- GitHub: https://github.com/infraWS
- Repository Issues: https://github.com/infraWS/sourceplane-mcp/issues

If possible, include:
- affected version
- reproduction steps
- expected behavior
- actual behavior
- potential impact

## Supported Versions

Currently supported:
- latest release only
